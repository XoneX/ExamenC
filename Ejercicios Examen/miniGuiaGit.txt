Si se ha creado algún archivo o carpeta nuevo que no está en el repositorio remoto,
hay que ejecutar:
git add -A

Siempre hay que hacer el commmit para que se guarden los cambios realizados:
git commit -am "Mensaje del Commit"
 
Y por último simplemente quedaría subirlo al repositorio remoto:
git push origin master



Para limpiar el historial de commits del git:
# Check out to a temporary branch:
git checkout --orphan TEMP_BRANCH

# Add all the files:
git add -A

# Commit the changes:
git commit -am "Initial commit"

# Delete the old branch:
git branch -D master

# Rename the temporary branch to master:
git branch -m master

# Finally, force update to our repository:
git push -f origin master
